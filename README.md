# uni-tour
对shopro-uniapp的二次开发  
shopro-uniapp链接：https://github.com/ITmonkey-cn/shopro-uniapp  
项目介绍：类似携程旅行的旅行网站，平台提供丰富的旅游产品，包括旅票、门票、餐饮、酒店、民宿、旅游团等。  
技术栈：Vue2+uniapp+Uview+node
